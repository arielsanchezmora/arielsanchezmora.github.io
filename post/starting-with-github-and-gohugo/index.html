Starting with Github and GoHugo

@arielsanchezmor · Sep 17, 2018 · 1 min read

So a way to get a blog hosted for free but still using your domain is to use a Git hosting service. I host blogs.arielsanchezmora.com on GitHub Pages, but Gitlab also offers the same thing.

This blog walks you through making a blog from scratch on GitLab using Windows as your main workstation. At some point I may create one from Linux.

1) ssh keys
2) Gitlab Pages link
3) GoHugo.io
4) Picking a theme
5) Getting started with MarkDown

First, to get Linux-like commands in Windows we’ll download Git for Windows. It asks a LOT of questions. I’ll put screenshots on what to choose soon, but you definitely want OpenSSL to be installed.

Second, we will create a new SSH key on our Windows computer. Look for the Git Bash icon, this will bring a Linux console. I had written this blogpost a while ago http://learning-in-it.blogspot.com/2016/11/creating-ssh-private-keys-advice-as-of.html
